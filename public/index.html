Free Steam Market JSON-HTTP-REST-API

Welcome to this free steam market api, hosted by zeit.co

API v1 GET

names : the market hash name of the item
appid : the app id of the game (https://steamdb.info/apps/)
currency : the currency id of the displayed price

Example: /api/v1?names=MP9 | Storm (Minimal Wear),Sawed-Off | Origami (Well-Worn)&appid=730¤cy=1
